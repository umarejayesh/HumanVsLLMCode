This repository provides the complete dataset used in our empirical study on the runtime and memory performance of human-written and LLM-generated code. The dataset includes 142 algorithmic problems from LeetCode and Codeforces, along with paired human-written and LLM-generated C++ solutions. For each solution, runtime and memory usage are reported under identical execution conditions. The dataset is released to support reproducibility and future research in AI-assisted programming.
